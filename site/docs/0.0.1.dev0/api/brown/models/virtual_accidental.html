VirtualAccidental

A logical accidental descriptor.

These are used both in Pitch manipulations and in graphical Accidental objects.

__init__ (value) permalink view source

arguments:

value (str or int or None): A description of the accidental.

  • 'f' or -1: Flat
  • 'n' or 0: Natural (Explicit)
  • 's' or 1: Sharp
  • None : No accidental, value depends on context.
