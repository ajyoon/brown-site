RepeatingMusicTextLine (MusicText, HorizontalSpanner)

A spanner of repeating music text over its length.

All of these spanners must be hori

arguments:

start (Point or init tuple): The starting point.

parent (GraphicObject): The parent of the starting point.

end_x (Unit): The end x position.

text (str, tuple, MusicChar, or list of these): The text to be repeated over the spanner, represented as a str (glyph name), tuple (glyph name, alternate number), MusicChar, or a list of them.

end_parent (GraphicObject): An optional parent of the end point. If omitted, the end position is relative to the main object.

font (MusicFont): The music font to be used. If not specified, the font is taken from the ancestor staff.

scale_factor (float): A hard scaling factor to be applied in addition to the size of the music font.
