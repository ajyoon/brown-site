BrushPattern (Enum)

An enumeration of pen stroke patterns.

Only the below enumerated patterns are currently supported, but if needed it is possible to implement gradient patterns and arbitrary custom patterns. See the following page from the relevant Qt docs: https://doc.qt.io/qt-5/qt.html#BrushStyle-enum
